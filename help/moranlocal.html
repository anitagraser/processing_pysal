Local Moran's I

Local Moran's I measures local autocorrelation quantitatively which results in n values of local spatial autocorrelation, one for each spatial unit.

Online docs: http://pysal.readthedocs.org/en/latest/users/tutorials/autocorrelation.html#local-moran-s-i

Inputs

Input

The polygon vector layer which should be analyzed.

Field

The field of the input layer which contains the values which should be analyized for clusters.

Contiguity

Define if queen or rook contiguity should be used to compute the weights.

Significance level

Select the significance level necessary for the output cluster to be identified as significant

Outputs

Local Moran's

The script outputs a polygon vector layer containing the original input features but appended by p-values MORAN_P and scatter plot quadrants

MORAN_Q values indicate quadrat location 1 HH, 2 LH, 3 LL, 4 HL (http://github.com/pysal/pysal/blob/master/pysal/esda/moran.py)

category scatter plot quadrant autocorrelation interpretation
high-high upper right (red) positive cluster I'm high and my neighbors are high
high-low lower right (pink) negative outlier I'm a high outlier among low neighbors
low-low lower left (med. blue) positive cluster I'm low and my neighbors are low
low-high upper left (light blue) negative outlier I'm a low outlier among high neighbors

http://www.biomedware.com/files/documentation/spacestat/Statistics/LM/Results/Interpreting_univariate_Local_Moran_statistics.htm

p_sim

String representation of the numpy array of resulting p-values for further usage in other Processing tools
