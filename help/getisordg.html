Getis and Ord's G

Getis-Ord global spatial autcorrelation.

Results are printed to the Python Console.

Online docs: http://pysal.readthedocs.org/en/latest/users/tutorials/autocorrelation.html#getis-and-ord-s-g

Inputs

Input

The polygon vector layer which should be analyzed.

Field

Attribute that will be tested for global spatial autcorrelation.

Contiguity

Define if queen or rook contiguity should be used to compute the weights.

Outputs

Results are printed to the Python Console.

G
