Local G and G*

Getis-Ord local spatial association statistics.

Online docs: http://pysal.readthedocs.org/en/latest/users/tutorials/autocorrelation.html#local-g-and-g

Inputs

Input

The polygon vector layer which should be analyzed.

Field

Attribute that will be tested for spatial association.

Contiguity

Define if queen or rook contiguity should be used to compute the weights.

Outputs

Local G and G*

The script outputs a polygon vector layer containing the original input features but appended by additional derived attributes:

  • L_G, the local statistics
  • L_G_P psuedo-pvalues for each of the local statistics,
  • L_G_S binary indicator as to whether statistic is significant,
  • L_G_LL_HH indicator variable 0=not sigificant, 1=signficant low-low clustering, 2=significant HH clustering
