Moran's I

Moran's I measures the global spatial autocorrelation in an attribute y measured over n spatial units.

Results are printed to the Python Console.

Online docs: http://pysal.readthedocs.org/en/latest/users/tutorials/autocorrelation.html#moran-s-i

Inputs

Input

The polygon vector layer which should be analyzed.

Field

The field of the input layer which contains the values which should be analyized for clusters.

Contiguity

Define if queen or rook contiguity should be used to compute the weights.

Outputs

Results are printed to the Python Console.
